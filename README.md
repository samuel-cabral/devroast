# devroast

Cole seu código e receba um "roast" — uma análise brutalmente honesta (e divertida) da qualidade do seu código, com nota de 0 a 10.

> Projeto construído durante o evento **NLW** da [Rocketseat](https://rocketseat.com.br), nas aulas do evento.

## O que faz?

- **Cole seu código** no editor e clique em "roast my code"
- Receba uma **nota de 0 a 10** com análise detalhada dos problemas encontrados
- Ative o **roast mode** para máximo sarcasmo nas respostas
- Confira o **shame leaderboard** — um ranking dos piores códigos já submetidos

## Funcionalidades

- Editor de código com numeração de linhas e scroll sincronizado
- Análise de código com badges de severidade (critical, warning, good)
- Visualização de diff com sugestões de melhoria
- Syntax highlighting com tema escuro
- Score ring visual com gradiente

## Rodando localmente

```bash
pnpm install
pnpm run dev
```

Acesse `http://localhost:3000`.

---

Feito com Next.js, React, Tailwind CSS e TypeScript.
